# css选择器汇总

css选择器参考手册

# 最常用选择器

选择器 举例 例子描述 CSS
.class .wlk 选择class="wlk"的所有元素 1
#id #wlk 选择id="wlk"的元素 1
* * 选择所有元素 2
element p 选择所有 <p>元素 1
element,element h1,h2 选择所有 <h1>元素和所有<h2>元素 1
element element div p 选择<div> 元素内部的所有<p>元素 1
element>element div>p 选择父元素为 <div> 元素的所有 <p>元素 2
element+element div+p 选择紧接在 <div>元素之后的<p>元素 2
element1~element2 p~ul 选择前面有<p>元素的每个<ul>元素。 3

# 属性选择器

有无引号都可以，本表统一加引号。（个人建议加上引号，可以避免一些错误，比如[a^="#"]就必须加引号）。

选择器 举例 例子描述 CSS
[attr] p[class] 选择带有 class 属性所有<p>元素 2
[attr=value] p[class="wlk"] 选择class=“wlk”的所有<p>元素，就“wlk”一个class 2
[attr~=value] [class~="wlk"] 选择class中包含单词"wlk"的所有元素。比如“nihao wlk hhh” 2
[attr^=value] a[href^="#"] href属性以 "#" 开头的a元素 3
[attr$=value] img[src$=".jpg"] src属性以".jpg"结尾的img元素 3
[attr*=value] a[href*="baidu"] href属性包含“baidu”的a元素 3
[attr|=value] [lang|=zh] lang属性为“zh”或lang属性以“zh-”开头的所有元素 2

备注：属性选择器默认是区分大小写的，可以加i或者I来忽略大小写。比如[class="wlk" i]。

# 伪类选择器

选择器 举例 例子描述 css
:link a:link 选择所有未被访问的链接。 1
:visited a:visited 选择所有已被访问的链接。 1
:hover a:hover 选择鼠标指针位于其上的链接。 1
:active a:active 选择活动链接。 1
:focus input:focus 选择获得焦点的 input 元素。 2
:before p:before 在每个<p> 元素的内容之前插入内容。 2
:after p:after 在每个<p> 元素的内容之后插入内容。 2
:first-letter p:first-letter 选择每个<p> 元素的首字母。 1
:first-line p:first-line 选择每个 <p> 元素的首行。 1
:first-child p:first-child 选择p，它是父元素的第一个元素。 2
:first-of-type p:first-of-type 选择p，它是父元素的第一个p元素。 3
:last-child p:last-child 选择p，它是父元素的最后一个元素。 3
:last-of-type p:last-of-type 选择p，它是父元素的最后一个p元素。 3
:only-child p:only-child 选择p，它是父元素的唯一一个元素。 3
:only-of-type p:only-of-type 选择p，它是父元素的唯一一个p元素。 3
:nth-child(n) p:nth-child(2) 选择p，它是父元素的第2个元素。 3
:nth-of-type(n) p:nth-of-type(2) 选择p，它是父元素的第2个p元素。 3
:nth-last-child(n) p:nth-last-child(2) 同:nth-child。只是从最后一个子元素开始计算 3
:nth-last-of-type(n) p:nth-last-of-type(2) 同:nth-of-type。只是从最后一个子元素开始计算 3
:root :root 选择文档的根元素。 3
:empty p:empty 选择没有子元素的 p 元素（文本节点也没有）。 3
:target #news:target 选择当前活动的 #news 元素。(不好理解，看实例) 3
:enabled input:enabled 选择每个启用的 input 元素。 3
:disabled input:disabled 选择每个禁用的 input 元素 3
:checked input:checked 选择每个被选中的 input元素。 3
:not(selector) .wlk:not(p) 选择非 p 元素的每个"".wlk"元素。 3
:lang(language) p:lang(it) 选择带有以 "it" 开头的 lang 属性值的每个<p>元素。 2
::selection ::selection 选择被用户选取的元素部分。 3
